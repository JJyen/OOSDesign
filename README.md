# OSS Design Project

## 실행환경
- 사용 편집기: Visual Studio Code
- 런타임 환경: Node.js (설치 필요)

## 실행방법
1. 프로젝트 파일로 이동합니다.
   ```bash
   cd oss-design
   ```
2. 다음과 같이 실행 명령어를 입력합니다.
   ```bash
   npm start
   ```

## 주의사항
1. open api 개인 키가 입력된 .env 파일이 깃허브에는 업로드되어있지 않습니다. **LMS 과제란에 제출된 프로젝트 파일**을 사용해주세요.
